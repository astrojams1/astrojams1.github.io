What brings you here?

This website is made by tweaking the minimum number of variables possible from all basic settings. I beleive in Numinous Emergence: That which ought to be is manifest through a collective vison.

I have a cool and simple workflow for building this website. I have Atom to code and use its built-in git tools to push to my Github Page. I then auto-sync my GitHub Page with my personal domain (jham.es). When I make design tweaks, I push changes to prod and evaluate on the end product (rather than staging changes). Excuse the mess.
